What noise conditions were applied to Hornsea Project Three?

Hornsea Project Three (EN010080) had several noise-related conditions in its DCO:

Construction noise: Condition 19 required a Construction Noise Management Plan limiting daytime construction noise to 65 dB LAeq at the nearest sensitive receptor [1]. Piling activities were restricted to 07:00-19:00 Monday to Saturday [2].

Operational noise: The operational noise limit was set at 35 dB LA90 at the nearest residential property, measured as a 10-minute sample [1] [3].

Proposed Chat Layout

Collapsible sidebar with localStorage history, centered message thread, pinned filter chips above input, Perplexity-style source cards below answers

Key Elements

Sidebar (Left)

  • Brand + "New Chat" button
  • Conversation history grouped by date
  • Stored in localStorage, loaded on page init
  • Collapsible on mobile (hamburger toggle)
  • "Scope Filters" button at bottom

Message Thread (Center)

  • User messages right-aligned (bubble)
  • Assistant messages left-aligned (full width)
  • Inline [n] citation badges, clickable
  • Expandable source cards below each answer
  • Markdown rendering for formatting

Input Area (Bottom)

  • Pinned filter chips (removable)
  • Auto-growing textarea
  • Submit button + Enter to send
  • Disclaimer text below

Source Cards

  • Document title + reference number
  • Document type, project type, topic badges
  • Page range indicator
  • Click to expand: shows source text snippet
  • Link to original source URL
